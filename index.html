Co to jest Puppet?

Puppet - prosty, deklaratywny język

package { 'postgresql':
  ensure => 'installed',
}
service { 'postgresql':
  ensure  => 'running',
  require => Package['postgresql'],
}

Czym jeszcze jest Puppet?

  • Firma i największa społeczność
  • Ekosystem technologii
  • Najlepsza implemetacja DevOpsowego configuration management
  • Separacja prostych deklaratywnych manifestów (naturalne dla administratorów) od kodu imperatywnego (naturalne dla programistów)
  • Profesjonalne narzędzie do wykorzystania przy wielu projektach i w całej firmie
  • Bardzo dużo gotowego kodu, brak konieczności all-or-nothing

Jak wykorzystać w JavaScript?

Sposoby wykorzystania

  1. Przygotowanie środowisk produkcyjnych - dla backend JS lub frontend w izolowanym środowisku
  2. Przygotowanie środowisk testowych i deweloperskich
  3. Wspólpraca z innymi działami
  4. Symulowanie backendu i/lub provisionowanie środowisk
